# Consulta-ViaCep

O arquivo .env não está incluído no repositório porque ele contém credenciais sensíveis do meu banco de dados local. Por segurança, essas informações não devem ser expostas publicamente. Para testar o projeto, o avaliador pode criar seu próprio arquivo .env na pasta do projeto, com as variáveis DB_USER, DB_PASSWORD e DB_NAME ajustadas para o MySQL da própria máquina. O código já está preparado para usar esses valores automaticamente por meio do python-dotenv.
